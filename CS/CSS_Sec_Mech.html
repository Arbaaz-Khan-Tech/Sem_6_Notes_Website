Understanding Firewalls and Memory Protection

Firewalls

Types of Firewalls

Packet Filtering

  • Examines packet headers
  • Based on IP addresses and ports
  • Stateless operation

Stateful Inspection

  • Tracks connection state
  • More intelligent filtering
  • Better security

Application Layer

  • Deep packet inspection
  • Protocol-specific filtering
  • Highest security level

Next-Generation

  • Integrated security features
  • Advanced threat protection
  • Intrusion prevention

Circuit Level Gateway

  • Works at session layer
  • Validates TCP handshakes
  • Hides internal network

Software Firewall

  • Installed on individual devices
  • Customizable rules
  • Cost-effective solution

Hardware Firewall

  • Physical network device
  • Network-wide protection
  • High performance

Cloud Firewall

  • Cloud-based protection
  • Scalable solution
  • Managed service

Firewall Layers

1. Physical Layer

  • Hardware-based protection
  • Network segmentation
  • Physical access control

2. Data Link Layer

  • MAC address filtering
  • VLAN security
  • Switch port security

3. Network Layer

  • IP packet filtering
  • Routing security
  • Network address translation

4. Transport Layer

  • TCP/UDP port filtering
  • Connection tracking
  • Session management

5. Application Layer

  • Protocol-specific filtering
  • Content inspection
  • Application control

Mnemonic: "Please Do Not Throw Away Pizza"

Memory Protection

Address Space Protection

Key Features:

  • Virtual memory separation
  • Page table protection
  • Memory segmentation
  • Address space layout randomization (ASLR)

Mnemonic: "Memory Protection = Virtual + Pages + Segments + Random"

Access Control

Protection Mechanisms:

  • Read/Write/Execute permissions
  • Memory access control lists
  • Process isolation
  • Privilege levels

Mnemonic: "Access Control = RWE + ACL + Isolation + Privilege"

Buffer Protection

Security Measures:

  • Stack canaries
  • Data execution prevention (DEP)
  • Address space layout randomization (ASLR)
  • Control flow integrity

Mnemonic: "Buffer Protection = Canary + DEP + ASLR + CFI"

⚠️ Very Important to Remember!

• Firewalls: Multiple layers of protection

• Memory: Virtual memory and access control

• Layers: Physical to Application

• Protection: Multiple mechanisms working together
