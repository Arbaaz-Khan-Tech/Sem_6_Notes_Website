Cheat Sheet: Data Analytics & Visualization

(Based on Mumbai University Syllabus & Previous Papers)

🔄Data Analytics Lifecycle

6 Key Phases:

  1. Discovery – Define problem, stakeholders, resources.
  2. Data Preparation – Clean, transform, load data (ETL).
  3. Model Planning – Choose algorithms (regression, clustering).
  4. Model Building – Train/test models.
  5. Communication – Visualize results (dashboards, reports).
  6. Operationalize – Deploy solutions.

📈Regression Analysis

Linear Regression:

  • Equation: y = β₀ + β₁x
  • Assumptions: Linear relationship, homoscedasticity, normality of residuals.

Logistic Regression:

  • Used for binary classification (e.g., yes/no).
  • Output: Probability (0 to 1) via sigmoid function.

Stepwise Regression:

  • Forward: Add predictors one by one.
  • Backward: Remove least significant predictors.

Key Formulas:

  • Correlation (r): Measures linear relationship (-1 to 1).
  • Regression Coefficients: β₁ = r·σy/σx, β₀ = ȳ - β₁x̄

⏳Time Series Analysis

Components:

  • Trend (long-term movement)
  • Seasonality (repeating patterns)
  • Noise (random fluctuations)

Models:

  • ARIMA(p,d,q): p: Autoregressive lags, d: Differencing order, q: Moving average terms.
  • ARMA: ARIMA without differencing (for stationary data).

Box-Jenkins Methodology:

  1. Identify model (ACF/PACF plots).
  2. Estimate parameters.
  3. Diagnose residuals.

📝Text Analytics

TF-IDF:

  • Term Frequency (TF): Term count in doc / Total terms in doc.
  • Inverse Document Frequency (IDF): log(Total docs / Docs containing term).
  • TF-IDF = TF × IDF.

Sentiment Analysis Methods:

  • Lexicon-based (e.g., VADER).
  • Machine learning (e.g., Naïve Bayes, LSTM).

📊Data Visualization

Python Libraries:

  • Matplotlib: Basic plots (line, bar, scatter).
  • Seaborn: Statistical visuals (heatmaps, boxplots).
  • Pandas: Quick plots (df.plot()).

R Libraries:

  • ggplot2: Layered grammar of graphics.
  • plotly: Interactive plots.

Common Plots:

  • Histogram: Distribution of numerical data.
  • Boxplot: Outliers, quartiles.
  • Scatterplot: Correlation between variables.

🐍Python/R Libraries

Pandas:

  • Key features: DataFrames, groupby, merge, pivot_table.

NumPy vs. Pandas:

  • NumPy: Numerical arrays, math operations.
  • Pandas: Labeled data, handling missing values.

Data Import/Export in R:

  • Read: read.csv(), read_excel().
  • Write: write.csv().

📊Statistical Tests & EDA

Exploratory Data Analysis (EDA):

  • Summary statistics (mean, median).
  • Visualizations (histograms, scatterplots).
  • Handle missing values (drop/impute).

Residual Analysis:

  • Check for patterns (should be random).
  • Q-Q Plot: Verify normality.

⚡Key Differences

Linear Regression Logistic Regression
Predicts continuous values Predicts probabilities (0/1)
Uses MSE loss Uses log-loss
Matplotlib Seaborn
Low-level, customizable High-level, statistical focus
ARMA ARIMA
For stationary data Includes differencing for non-stationary data

🧮Quick Formulas

  • Regression Lines: y on x: y - ȳ = r·σy/σx (x - x̄). x on y: Swap x and y.
  • Correlation (r): r = Cov(x,y) / (σxσy).
  • TF-IDF Example: Doc: "The cat sat" → TF("cat") = 1/3. IDF("cat") = log(100/10) = 1 (if 10 docs contain "cat" out of 100).

📝Short Notes

  • Generalized Linear Model (GLM): Extends linear regression for non-normal distributions (e.g., logistic for binary data).
  • Box-Jenkins Intervention Analysis: Adjusts time series models for external events (e.g., COVID impact on sales).
  • Clickjacking: Attack tricking users to click hidden UI elements.
