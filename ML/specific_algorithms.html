McCulloch-Pitts Model (1943)

The first mathematical neuron ‚Äî binary logic gates made simple!

Key Concepts:

  • Binary Inputs/Outputs: Only accepts and outputs 0 (inactive) or 1 (active)
  • Fixed Weights: No learning ‚Äî weights are manually set
  • Threshold Logic: Fires if weighted sum ‚â• threshold (Œ∏)

Formula:

\[y = \begin{cases} 1 & \text{if } \sum_{i} w_i x_i \geq \theta \\ 0 & \text{otherwise} \end{cases}\]

Hebbian Learning (1949)

"Neurons that fire together, wire together!"

Key Concepts:

  • Unsupervised Learning: No labels needed ‚Äî learns from input patterns
  • Local Updates: Synapses strengthen based on co-activation
  • Biological Basis: Models how real neurons adapt (neuroplasticity)

Formula:

\[\Delta w_{ij} = \eta \cdot x_i \cdot y_j\]

where:

  • \(\Delta w_{ij}\) is the weight change
  • \(\eta\) is the learning rate
  • \(x_i\) is the input activation
  • \(y_j\) is the output activation

Support Vector Machines (SVM)

"The maximal margin classifier ‚Äî finding the best decision boundary"

Key Concepts:

  • Hyperplane: Decision boundary \(w^T x + b = 0\)
  • Margin Maximization: Widest possible gap between classes
  • Kernel Trick: Maps data to higher dimensions (e.g., RBF kernel)

Optimization Problem:

\[\min_{w,b} \frac{1}{2} \|w\|^2 \text{ subject to } y_i(w^T x_i + b) \geq 1 \text{ for all } i\]

where:

  • \(w\) is the weight vector
  • \(b\) is the bias term
  • \(x_i\) are the training points
  • \(y_i\) are the class labels (-1 or 1)
