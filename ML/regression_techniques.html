Linear Regression

Basic Equation

\[ \hat{y} = \beta_0 + \beta_1x_1 + \beta_2x_2 + \cdots + \beta_nx_n \]

Goal: Minimize Mean Squared Error (MSE)

\[ \min \sum(y_i - \hat{y}_i)^2 \]

Ridge vs. Lasso

Ridge Regression

\[ \min(\sum(y_i - \hat{y}_i)^2 + \lambda\sum\beta_j^2) \]

Lasso Regression

\[ \min(\sum(y_i - \hat{y}_i)^2 + \lambda\sum|\beta_j|) \]

Linear vs Logistic Regression

Linear Regression

\[ y = \beta_0 + \beta_1x_1 + \beta_2x_2 + \cdots + \beta_nx_n \]

Characteristics:

  • Predicts continuous values
  • Uses MSE as loss function
  • Output range: (-∞, +∞)
  • No activation function

Use Cases:

  • House price prediction
  • Stock price forecasting
  • Temperature prediction
  • Sales forecasting

Logistic Regression

\[ p(y=1) = \frac{1}{1 + e^{-(\beta_0 + \beta_1x_1 + \beta_2x_2 + \cdots + \beta_nx_n)}} \]

Characteristics:

  • Predicts probabilities
  • Uses cross-entropy loss
  • Output range: (0, 1)
  • Sigmoid activation

Use Cases:

  • Spam detection
  • Disease diagnosis
  • Credit approval
  • Customer churn

Regularization Methods Comparison

Linear Regression

\[ \min \sum(y_i - \hat{y}_i)^2 \]

Characteristics:

  • No regularization
  • Can overfit
  • Simple implementation
  • Fast computation

Ridge Regression (L2)

\[ \min(\sum(y_i - \hat{y}_i)^2 + \lambda\sum\beta_j^2) \]

Characteristics:

  • L2 penalty term
  • Shrinks coefficients
  • Never zero coefficients
  • Good for multicollinearity

Lasso Regression (L1)

\[ \min(\sum(y_i - \hat{y}_i)^2 + \lambda\sum|\beta_j|) \]

Characteristics:

  • L1 penalty term
  • Feature selection
  • Can zero coefficients
  • Good for sparse data

Key Differences

  • Ridge shrinks coefficients but never to zero
  • Lasso can force coefficients to exactly zero
  • Ridge is better for multicollinearity
  • Lasso is better for feature selection

When to Use

  • Use Ridge when all features are important
  • Use Lasso when you suspect sparse features
  • Use Elastic Net for a balance of both
  • Always cross-validate λ parameter

Common Pitfalls

  • Not scaling features before regularization
  • Choosing λ without cross-validation
  • Ignoring multicollinearity
  • Overlooking feature importance
