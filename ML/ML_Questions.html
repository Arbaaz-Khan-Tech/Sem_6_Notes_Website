Machine Learning Cheat Sheet

Based on Mumbai University Syllabus & Previous Year Papers

1. Core Concepts

Supervised

Labeled data (e.g., regression, classification)

Unsupervised

No labels (e.g., clustering, PCA)

Reinforcement

Learns via rewards (e.g., Q-learning)

2. Regression

Linear Regression

\[Y = \beta_0 + \beta_1 X\]

  • Least Squares Method: Minimizes \(\sum (Y_i - \hat{Y_i})^2\)
  • Assumptions: Linear relationship, homoscedasticity, normality of residuals

Regularization

Ridge (L2)
  • Penalizes \(\beta^2\)
  • Prevents overfitting
Lasso (L1)
  • Penalizes \(|\beta|\)
  • Feature selection

3. Neural Networks

Perceptron Model

\[y = f(\sum w_i x_i + b)\]

Activation Functions

  • Binary Step: \[f(x) = \begin{cases} 1 & \text{if } x \geq 0 \\ 0 & \text{otherwise} \end{cases}\]
  • Sigmoid: \[f(x) = \frac{1}{1 + e^{-x}}\]
  • ReLU: \[f(x) = \max(0, x)\]

4. Model Evaluation

Metrics

Accuracy

\[\frac{TP + TN}{TP + TN + FP + FN}\]

Precision

\[\frac{TP}{TP + FP}\]

Recall

\[\frac{TP}{TP + FN}\]

F1-Score

\[2 \times \frac{\text{Precision} \times \text{Recall}}{\text{Precision} + \text{Recall}}\]
