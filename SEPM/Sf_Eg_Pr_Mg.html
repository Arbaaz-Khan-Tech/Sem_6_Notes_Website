2. COCOMO Cost Estimation

Basic COCOMO Formula

Effort = a × (KLOC)ᵇ (Person-Months)

Time = c × (Effort)ᵈ (Months)

Organic

a = 2.4, b = 1.05

Small teams, familiar environment

Semi-Detached

a = 3.0, b = 1.12

Medium teams, mixed experience

Embedded

a = 3.6, b = 1.20

Large teams, complex environment

Mnemonic 1: "COCOMO = Count Of Code, Organize Management Overhead"

Mnemonic 2: "Project Types: Organic (2.4), Semi-Detached (3.0), Embedded (3.6)"

3. Project Scheduling (CPM/PERT)

Critical Path Method (CPM)

"Finds the longest path (critical path) to determine project duration."

Early Start (ES)

Earliest possible start time

ES = Max(EF of predecessors)

Late Finish (LF)

Latest possible finish time

LF = Min(LS of successors)

Slack Time

Tasks that can delay without affecting timeline

Slack = LF - EF

Program Evaluation Review Technique (PERT)

"Uses probabilistic time estimates (Optimistic/Pessimistic/Most Likely)."

Expected Time Formula

Expected Time = (O + 4M + P) / 6

O: Optimistic

Best-case scenario

Minimum time needed

M: Most Likely

Normal scenario

Most probable time

P: Pessimistic

Worst-case scenario

Maximum time needed

Mnemonic: "CPM = Critical Path, PERT = Probabilistic Time!"
